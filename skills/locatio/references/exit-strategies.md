# Exit Strategies Reference

## Paths Out of On-Hold

### 1. Knowledge → 30. KNOWLEDGE

**Description**: Convert learned content into permanent knowledge notes

**Process**: Study held notes → create knowledge note(s) in `30. KNOWLEDGE/` → archive via memorio

**When to use**:
- After understanding the content
- When ready to distill key insights
- Creating reference material

**User phrases**:
- "이제 다 학습함"
- "정리해서 KNOWLEDGE로"
- "지식화 완료"

### 2. Project → elevatio → 51. Active

**Description**: Convert notes into an active project

**Process**: `/elevatio` with topic → move notes to project → archive HOLD.md

**When to use**:
- Ready to work on the topic
- Have clear project goals
- Time/resources available

**User phrases**:
- "이제 프로젝트로 시작"
- "프로젝트화"
- "elevatio로 이동"

### 3. Archive → memorio → 54. Archive

**Description**: Archive without further processing

**Process**: Create archive record → move topic folder to archive → update MOC

**When to use**:
- No longer interested in topic
- Content became outdated
- Found better resources elsewhere

**User phrases**:
- "더 이상 필요 없음"
- "아카이브"
- "정리"
