# Hold Reasons Reference

## Categories

### purgatio_bypass
**Description**: Notes don't fit standard purgatio destinations

**When to use**:
- Expert/specialized notes that need more context
- Notes that require learning before processing
- Grouped notes from same source (e.g., blog series)

**User phrases**: "갈 곳이 없음", "purgatio로 처리하기 애매함"

### not_ready
**Description**: Notes require learning/study before processing

**When to use**:
- Technical content that needs understanding first
- Complex topics requiring research
- Content with steep learning curve

**User phrases**: "학습 필요", "아직 공부 안 함"

### exploring
**Description**: Still exploring the topic area

**When to use**:
- New domain being investigated
- Not sure if topic will be useful
- Gathering information phase

**User phrases**: "아직 탐색 중", "더 알아봐야 함"

### overload
**Description**: Immediate projectization would be overwhelming

**When to use**:
- Too many other active projects
- Need to reduce cognitive load
- Want to focus on current priorities

**User phrases**: "지금 프로젝트 너무 많음", "일단 보류"
